🚦 Road Accident Analysis Dashboard (Excel)

This project presents a Road Accident Analysis Dashboard created in Microsoft Excel to analyze accident trends, casualty severity, and road conditions.
The dashboard helps identify patterns in accidents and provides insights that can support road safety improvements.

---

🎯 Project Objective

The goal of this project is to analyze accident data and understand key factors contributing to road casualties.

Key focus areas:

- Total number of casualties
- Accident severity analysis
- Casualties by vehicle type
- Road and weather conditions
- Monthly accident trends

---

📊 Key Dashboard Features

Casualty Overview

- Total number of casualties
- Fatal, Serious, and Slight severity breakdown

Accident Insights

- Casualties by vehicle type
- Casualties by road type
- Casualties by surface conditions

Trend Analysis

- Current year vs previous year casualty trends
- Monthly accident patterns

Environmental Factors

- Light conditions (Daylight vs Darkness)
- Weather conditions
- Area type (Urban vs Rural)

Interactive Filters

- Accident year
- Weather conditions
- Wind conditions

---

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization
- Data Analysis

---

💡 Key Insights

- Slight severity accidents account for the majority of cases
- Most accidents occur on single carriageway roads
- Daylight conditions record a significant number of incidents
- Urban areas show higher accident frequency compared to rural areas
